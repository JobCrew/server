🔧 주요 API 엔드포인트

  • SSE 연결: GET /api/notify/sse/{memberName}
  • 하트비트: POST /api/notify/heartbeat
  • 채팅 알림 발송: POST /api/test/notifications/send-chat
  • 시스템 알림 발송: POST /api/test/notifications/send-system
  • 테스트 사용자 목록: GET /api/test/notifications/test-users
  • 연결 상태 확인: GET /api/test/notifications/status/{memberName}
